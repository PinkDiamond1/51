Examples

Create a new Customer
curl https://api.51bank.io/customers -Xpost \ -d firstName=Alice \ -d lastName=McCartney \ -d email=alice.mccartney@gmail.com \ -d nationality=GB \ -d dateOfBirth="1985-08-23" \ -d address='/GB/220 cleveland road/london/w13 9pn'
Modify an existing customer
curl https://api.51bank.io/customers{/uniqueId}} -Xpatch \ -d firstName=Alicia
Delete a customer
curl https://api.51bank.io/customers{/uniqueId} -Xdelete
List one customer
curl https://api.51bank.io/customers{/uniqueId}
List all customers
curl https://api.51bank.io/customers/
Create a new Account
curl https://api.51bank.io/accounts -Xpost \ -d customerId={{uniqueId}} \ -d description="A little saving account"
Modify an existing account
curl https://api.51bank.io/accounts/{{uniqueId}} -Xpatch \ -d description="A mini saving account"
Delete an account
curl https://api.51bank.io/accounts/{{uniqueId}} -Xdelete
List one account
curl https://api.51bank.io/accounts/{{uniqueId}}
List all accounts
curl https://api.51bank.io/accounts/

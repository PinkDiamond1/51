Syntax Errors

Expecting CURL Command

The command that was typed does not begin with "CURL"
Example: 
hello

Abrupt New Line

The command is missing the backslash (\) character at the end of a line. The backslash character is required to indicate that the command continues on the next line.

Duplicate URL Definition

More than one URL was found in the command, leaving the parser unable to continue

Unsupported Option

The CURL parser does not support a specified option. This might be a valid option (e.g. -s) or an invalid option. Current option support is very limited to few options '-d', '-x'

Post/Patch without data

The command is asking either to post or to patch but without data. The current version of the CURL Parser does not allow it.

Invalid Http Method

The requested method is not valid
